Python test practical

Specification

FUNCTION: overlap(rectangle1, rectangle2)

INPUT: two rectangles.

OUTPUT: rectangle representing overlap between the two input rectangles.

Rectangle is defined as a list of four points: [x0, x1, y0, y1] where:

  • 0 < x0 < x1
  • 0 < y0 < y1

Task

In pairs...
  • Create a repository 
    hg init
  • Create source file, rectangle.py 
    def overlap(a, b):
    pass
  • Create test file, test-overlap.py 
    from rectangle import overlap

def test_coincident():
    a = [0, 2, 0, 2]
    b = a
    expected = a
    actual = overlap(a, b)
    assert expected == actual
  • Write more tests
  • Add to repository 
    hg add rectangle.py test-overlap.py
  • Write code to make a test pass
  • Run tests 
    nosetests test-overlap.py
  • Commit 
    hg commit
  • Repeat until all tests pass
  • When happy push to a BitBucket repository

Example test 

from rectangle import overlap

def test_something():
    a = [3, 6, 3, 6]
    b = [0, 4, 0, 4]
    expected = [3, 4, 3, 4]
    actual = overlap(a, b)
    assert expected == actual
